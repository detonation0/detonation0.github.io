About This Site


My name is Mathew Ruff, I'm a web developer, guitarist, and gamer living in Tempe, Arizona. I really enjoy developing for the web and am currently working for Arizona State University on the Salesforce Development Team. This site was initially developed as a class assignment that challenged the students to create a Windows Azure Virtual Machine and host a website from it. Since then, the site has been evolved into my own project. Here you will find links to my professional information, original and cover music, technology interests, and interesting information I find on the web.

I've really enjoyed developing this site, it's become a fun project that I didn't think would evolve beyond a class assignment into something I would actually use. I love to learn and I'm always looking for the newest technology to perfect and implement into my work. I'm going to keep improving the site beyond what you might see currently, adding more content whenever possible.
