About This Site

My name is Mathew Ruff, I'm a web/salesforce developer, guitarist, and gamer living in Tempe, Arizona. I really enjoy developing for the web and am currently working as a Salesforce Developer for Arizona State University on the Salesforce Enterprise Development Team. This site was initially developed as a class assignment that challenged the students to create a Windows Azure Virtual Machine and host a website from it. Since then, the site has been evolved into my own project. Detonation0 has always been a handle I went by on the web whether it was a gaming name, Twitter handle, or username. So naturally I've adapted a sort of bomb theme on the site just for fun.

I've really enjoyed developing this site, it's become a fun project that I didn't think would evolve beyond a class assignment into something I would actually use. I love to learn and I'm always looking for the newest technology to perfect and implement into my work. I plan on continuing to improve and add to the site in the future.
