Digital Portfolio

DETONATIONBLOG

My personal site; it utilizes Twitter's Bootstrap 3 mobile framework for a fully responsive layout. It was built from scratch using static HTML/CSS/JavaScript/jQuery. I use it as a blog as well as a place to post my original music and professional information. I continue to update it whenever I have the chance with new features, posts, and technologies I've learned.

DETONATIONBLOG [Mobile]

Because the majority of what I build for the web I make responsive, this is the mobile version of DETONATIONBLOG. Automatically resizes to fit tablets and mobile devices. Uses Twitter Bootstrap 3 for a responsive layout.

Old Blog [Mobile]

This was the original DETONATIONBLOG. It was built on jQuery's Mobile Framework. Because of this, there had to be a mobile version of each page which was harder to maintain. So I built a new responsive site to cut down on the number of pages and align with current development trends.

Charlene Ruff's Personal Site

A professional site I built for my wife. She's a recent graduate from Arizona State University with a Master's of Science in Design with a concentration in Industrial Design.

Bennett's Carpet Cleaning Redesign

The Bennett's Carpet Cleaning redesign was built in my undergraduate capstone class at ASU. We used Wordpress to create a site where their customers could visit and get information, schedule consultations, and buy products. Still in use by the company.

NightLife Finder

NightLife Finder is a night life entertainment finder we built in Visualforce using Salesforce as the backend. The app allowed the user to look up entertainment (bars and local bands playing) and get directions using device location and Google Maps.

NightLife Finder [Mobile]

The mobile version of NightLife Finder was built on jQuery's Mobile Framework.

Metal Guitar Depot

An E-commerce site project built for my CIS425 class using PHP. Not hosted anywhere but you can view the code files on my Github

MyASU Service Center

A new tab on the MyASU student portal I built for the University using Visualforce and Apex. The new tab offers students a place for 24/7 live chat, service case creation and maintenance, and the ability to search the ASU knowledge base for answers to their questions.
